# Moscow Walks Bot

Чат-бот «Прогулки по Москве» для мессенджера MAX. 12 пеших маршрутов с фильтрами, пошаговой навигацией и избранным.

## Что в репозитории

- `bot.py` — основной код бота
- `routes_database.json` — база из 12 маршрутов
- `requirements.txt` — зависимости Python
- `amvera.yml` — конфиг для деплоя на Amvera

## Деплой с телефона

1. Загрузите все 4 файла в GitHub-репозиторий (через приложение GitHub или веб-интерфейс)
1. Зарегистрируйтесь на amvera.ru
1. Создайте новый проект → «Подключить GitHub» → выберите репозиторий
1. В разделе «Переменные окружения» добавьте `MAX_BOT_TOKEN` с вашим токеном от MasterBot
1. Нажмите «Собрать и запустить»

Через 1–2 минуты бот заработает в MAX.

## Локальный запуск

```bash
pip install -r requirements.txt
export MAX_BOT_TOKEN="ваш_токен"
python bot.py
```

## Команды бота

- `/start` — главное меню
- `/help` — справка

Всё остальное — через кнопки.
